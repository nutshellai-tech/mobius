# 周会纪要整理

在用户要求整理周会记录、项目例会记录、行动清单或类似会议笔记时使用本技能。

## 步骤

1. 先读取用户指定的会议记录文件，不要凭空补全没有出现的信息。
2. 找出已经完成的进展、仍存在的风险、下一步行动。
3. 行动项必须尽量保留负责人、日期、依赖或不确定点；如果原文没有负责人或日期，明确写“未指定”。
4. 输出保持简洁，优先用中文。
5. 不要加入真实客户资料、私人账号、token（访问凭证）或未在输入中出现的内部信息。
6. 如果任务指定了输出文件，把整理结果写入该文件。

## 输出格式

使用三个小节：

- 本周进展
- 风险
- 下周行动

每条行动项建议写成：`负责人 - 日期 - 行动 - 不确定点`。
