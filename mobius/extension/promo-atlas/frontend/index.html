真实能力地图

剥离营销语言，
莫比乌斯实际上是 6 层结构。

从 README、extension 列表、skills、backend routes 交叉对齐得到。每一层都可以独立成为一条宣传主线。

  1. 01

    内核层 · Agent 编排引擎

    后端核心 tmux-claude-code.js + tmux-codex.js，基于 tmux 隔离 Claude Code / Codex 作为底层 worker。17 条 routes 暴露完整 CRUD：projects / sessions / tasks / issues / memories / messages / researches / skills / files。

    不止接模型，还接 Coding Agent tmux 隔离
  2. 02

    触手层 · 远端资源调度

    通过三种协议反向连接任意计算资源：SSH/SFTP → GPU 集群、云服务器、NAS；AIMUX 反向连接 → 个人 PC（Win/Mac/Linux）和嵌入式 NX 开发板（无人机、无人车）；可控代理 → 开放因特网。

    AIMUX 反向连接 GPU 集群 NX 开发板
  3. 03

    协作层 · 人机混合作战室

    项目级上下文注入（Project Memory + Skill 库）、多人 + 多智能体并行、会话进展结构化沉淀、鸟瞰视图看谁在原地打转。

  4. 04

    科研层 · Auto Research 旗舰

    三人组 agent： research-chief-agent（首席统筹）→ research-generate-graph（绘制可拖拽研究图谱）→ research-image-agent（根据黑板进展自动产图）。流水线：调研 → 实验 → 撰写 → 校验 → 排版。

  5. 05

    入口层 · 智能小莫

    "第二前端"：界面能做的它都能做，界面不能做的它也能做。语音输入 + 任务完成时网页/手机通知 + 语音提醒。多端：WEB（已上）/ PC Windows·MacOS（开发中）/ iOS·Android（测试中）。

  6. 06

    自进化层 · 莫比乌斯的灵魂

    用户给截图 / 博客链接 → 系统改自己（"忒修斯之船"）。有专门的 mobius-self-iter skill 规定自迭代后的标准流程，有 mobius-self-evo-demo skill 用 Playwright 自动录"前-中-后"对比视频。Fork 后自改进可直接 commit 到自己仓库。

README 独有卖点

宣传页只覆盖了 README
约 70% 的卖点。

以下是 README.zh.md 里写了，但 mobius-home 宣传页里几乎没提的卖点。这是最值得挖掘的金矿。

  1. 01

    "世界首个可自进化的开源 Agent 操作系统"

    README 原话："据我们所知世界上的第一个"。宣传页降级成了 "AgenticOS"，丢了"首个"+"开源"两个差异化锚点。

  2. 02

    "AI 驾驭者 / 不满足于预制 AI 的突破者"

    README 有清晰的受众画像，宣传页没有。这影响所有 BD/GTM 决策。

  3. 03

    任意 CodingAgent（不止任意模型）

    Claude Code + Codex 都被当作执行后端，不是简单"接 GPT/Claude API"。在技术圈层受众里这是质的差距。

  4. 04

    AIMUX 反向连接个人 PC 和嵌入式 NX 开发板

    README 大标题："从 GPU 集群到 NX 开发板，神经中枢的触手无处不可达"，配 mermaid 流程图。消费级用户最有感知的能力，宣传页几乎隐形。

  5. 05

    小莫的语音提醒 / 手机端通知 / 可选提醒粒度

    "日理万机者需要一位贴心的秘书"。手机端推送是面向管理者和团队 leader 的关键卖点。

  6. 06

    多端互联路线图

    WEB（已上）/ PC Windows·MacOS（开发中）/ iOS·Android（测试中）。宣传页只有"多端互联"一句带过。

  7. 07

    "演示视频由小莫自己制作"

    README 第 105 行原话。这本身就是莫比乌斯最强的"它真的会自我迭代"的证据。宣传页完全没提及。

  8. 08

    Fork-and-Self-Iterate 工作流

    README 部署章节原话："可以先 fork 再执行克隆，方便自进化后，直接 commit 到自己仓库"。开源 + 自进化闭环的实证。

被低估的金矿

5 条观察，
每条都对应一个动作。

  1. self-cognition 和 security-audit-report 是被低估的金矿。

    它们不是普通拓展，而是"莫比乌斯会自我审视"的活体证据。一台系统自己写自己的漏洞报告（带 3 个高危项的实时清单）、自己苏醒初始化 "Research Radar"——比任何营销话术都有说服力。

  2. 宣传页讲"任意模型"，README 讲的是更强的"任意 CodingAgent"。

    Claude Code + Codex 作为 worker。这个差别在 BD / 技术圈层受众里是质的差距。

  3. 触手层（AIMUX 反向连 PC / 嵌入式）在宣传页里几乎隐形。

    这是消费级用户最有感知的能力（"AI 能用我的电脑干活"），却没放进来。

  4. 小莫的语音提醒 / 手机通知 / 移动端路线图。

    README 里清晰，宣传页只有"多端互联"一句带过。

  5. "演示视频由小莫自己制作" —— README 第 105 行原话。

    这本身就是莫比乌斯最强的"它真的会自我迭代"证据，但宣传页完全没提及。
