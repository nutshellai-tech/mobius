aimux 是什么

aimux 是莫比乌斯与你这台电脑之间的反向连接通道。连上之后，智能体可以在你电脑上运行命令、修改代码、操作文件，也能调度多台机器一起干活。桌面端登录后会自动连上，通常无需手动操作。

identifier
这台电脑在 mobius 里的名字（自动生成，用于辨识）。
服务器
它当前连上的 mobius 中枢地址。
aimux 版本 / venv 路径 / 内置 Python
aimux 程序自身的运行环境，正常情况下无需关心。
允许 aimux 反向连接
下面的开关。关掉后智能体就不能再用这台电脑干活了；桌面端登录、工作台等其他功能不受影响。
本机环境
智能体调度任务时会参考的硬件信息（系统 / CPU / 内存等）。
最近日志
连接过程的运行记录，出现问题时用来排查。
