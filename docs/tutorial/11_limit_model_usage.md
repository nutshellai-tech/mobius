# 如何限制模型调用频率

‍

在「**管理中心 → 系统设置 → 模型创建限制**」里，可以**按模型**设定调用上限，防止某个模型被某个用户 / 全体刷爆额度。

## 1. 进入「模型创建限制」

- 右上角头像菜单 → **管理中心** →「**系统设置**」标签 → 找到「**模型创建限制**」卡片。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/adm-limits.jpg)

## 2. 两类设置

- **全局默认模型**：当项目没设默认模型、用户也没手选时，新建会话回落到的模型。
- **每个模型 4 个硬限制**（留空 = 不限制）：
  - **所有用户 5h** —— 全体用户合起来，每 5 小时最多 N 次
  - **所有用户 5m** —— 全体用户合起来，每 5 分钟最多 N 次（防瞬时洪峰）
  - **单用户 5h** —— 每个用户每 5 小时最多 N 次
  - **单用户 5m** —— 每个用户每 5 分钟最多 N 次

## 3. 超限会怎样

- 某个模型对你达到上限后，它会在**模型选择菜单里被标红禁用**（鼠标悬停可见剩余 / 上限）。
- 5 分钟级的限制会在窗口结束后自动恢复；5 小时级的要等下一个 5 小时窗口。
- 改完点「保存」即时生效，不需要重启。
