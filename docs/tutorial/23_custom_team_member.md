# 创建自定义团队成员

‍

预设的三个成员只是起点。你可以「添加」一个**完全自定义**的成员——不绑定任何预设 skill，按你写的职责自由发挥。

## 操作

- 点标签栏「**添加**」新增一个助理成员，可自由命名（例如「文献综述专员」）。
- 编辑它的**名称**和**目的 / 职责**。
- 把「**主 Skill**」保持为「**完全自定义**」（即不选预设 skill）。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/tut23-custom.jpg)

!!! note
    主 Skill 一旦选成某个预设 skill，就会与之**互斥锁定**（避免两个成员抢同一份 skill）。想让成员自由发挥就保留「完全自定义」，再用下方的 Skill / Memory 勾选控制它能看到哪些上下文。
