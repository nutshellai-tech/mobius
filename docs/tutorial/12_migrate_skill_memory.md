# 如何迁移 Skill 和 Memory（Skill 与 Memory 管理）

‍

Skill 和 Memory 可以**跨用户 / 跨项目复用**：在「**管理中心 → Skill 与 Memory 管理**」里把它们**导出**成文件，再在别处**导入**即可。也支持直接新建。

## 1. 进入「Skill 与 Memory 管理」

- 右上角头像菜单 → **管理中心** →「**Skill 与 Memory 管理**」标签。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/adm-migration.jpg)

## 2. 导出（迁出）

- 在列表里**勾选**要迁出的 Skill / Memory（可用「全选我的 / 全选当前分组」批量选）。
- 点右上角「**导出**」，下载得到一个文件。

## 3. 导入（迁入）

- 在**目标账号 / 目标项目**里打开同一个面板，点「**导入**」，上传刚才的文件即可迁入。
- 导入时若同名条目已存在，按提示选择覆盖或跳过。

## 4. 其它

- 「**新建我的 Skill / Memory**」可直接新建一条用户级（对所有项目生效）的 Skill 或 Memory。
- 想了解 Skill / Memory 的三种层级（内置 / 用户级 / 项目级）与日常使用，见 [使用技能与记忆](03a_using_skill_and_memory.md)。
