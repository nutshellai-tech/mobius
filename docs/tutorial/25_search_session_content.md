# 跨会话搜索：在所有会话里查找内容

‍

会话越开越多，想找回某句对话、某段代码或某个关键词？顶栏新增了**全局搜索**，一次性扫描所有会话的完整内容——包括你之前关掉的历史会话。

## 怎么做

1. 点顶栏 **「搜索」** 按钮（在 `+新建` 左侧），弹出搜索框。

    ![image](https://serve.nutshellai.cn/publish/auto/tutorial/25_search_01.jpg)

2. 输入关键词（至少 2 个字符；停下片刻会自动搜索，也可以直接回车）。

    ![image](https://serve.nutshellai.cn/publish/auto/tutorial/25_search_02.jpg)

3. 在结果里找到目标会话，**点击任意一条结果**，直接跳转到该会话的聊天页。

## 结果怎么读

每条结果是一张卡片，从左到右是它的**归属层级**：

- **项目 › Issue / Research › Session** —— 这句话来自哪个项目、哪个任务、哪个会话。
- 下方是**命中的原文片段**，关键词会**黄色高亮**，左侧小标签标注角色（用户 / 助手 / 工具）。
- 右上角是这条会话的**最近活跃时间**。

## 注意

- 默认只扫描**最近活跃的 200 个会话**，每个会话最多展示 3 条命中片段；结果太多时换更具体的关键词。
- 单个超大会话只搜最近约 1.5MB 的对话尾部（最常翻的就是最近的对话）。
- 你**只能搜到有权限查看**的会话：自己创建的，或所在项目对你可见的。
