# Connect to Mobius from Your Workstation or Laptop

A personal workstation or laptop can also connect directly to the Mobius system. There are two main ways:

- Connect via the **aimux + Python command line**
- Connect via the **desktop client**

This guide covers the first approach.

## Step 1: Open the connection guide

![Connection guide panel](https://serve.nutshellai.cn/publish/auto/tutorial/tut27-connect-guide.jpg)

## Step 2: Click the copy button to copy the first install command
