# 使用指南 - 屏蔽与当前项目无关的 Skill 和 Memory

‍

> 新建会话时，「**预览配置**」步骤会列出本次**将要注入**的全部 Skill 和 Memory（项目级 + 你的用户级）。无关的技能/记忆会干扰智能体、白烧 Token —— 在这里逐条**取消勾选**即可屏蔽，它们就不会进入本次会话。

### 1. 进入「预览配置」

在任务单（Issue）里点【+ 新会话】，填好名称和描述后，点【**下一步 · 预览配置**】。

### 2. 屏蔽无关 Skill

![image](https://serve.nutshellai.cn/publish/auto/tutorial/07_exclude_skills.jpg)

- **①** 预览列出本次将注入的全部 Skill。
- **②** 取消勾选与本项目无关的（例如一个纯代码项目里的「音乐生成 / 视频生成」技能），它就不会被注入；取消后该项变灰、计数变为 `Skill (4/6)`。
- **③** 也可用右上角的【全不选】一键清空，再只勾你需要的。

### 3. 屏蔽无关 Memory

![image](https://serve.nutshellai.cn/publish/auto/tutorial/07_exclude_memories.jpg)

- **①** Memory 列表同样可逐条勾选/取消。
- **②** 取消勾选过期或与本次无关的记忆（如「历史废弃接口清单」），避免误导智能体。

> 勾选状态会**快照固化**到本次会话：创建后该会话不再受后续新增/删除 Skill·Memory 的影响。需要长期屏蔽某项，请到对应的用户/项目设置里直接删除它。
