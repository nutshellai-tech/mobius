# 组建智能体研究团队

‍

进入一个研究课题后，点右侧「**新 Agent**」→「**创建 Agent 团队**」，打开团队配置面板。默认给出 3 个成员（1 首席 + 2 助理），逐个创建并自动启动；右侧 3D 画布展示团队棋盘。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/tut20-team.jpg)

## 增减成员

- **添加**：点标签栏末尾的绿色「**添加**」按钮，新增一个助理成员（上限 12 个）。
- **删除**：把鼠标悬停到某个助理标签上，点出现的 ✕ 即可移除该成员。
- **首席（chief）不可删除**：每个团队必须有且只有一个首席，负责拆解课题、整合结论。

## 每个成员可单独配置

- 名称、目的 / 职责、模型、语言、主 Skill、以及 Skill 与 Memory 的启用范围。
- 点底部「**创建并启动团队**」后，系统逐个创建会话、写入 Research Blackboard 团队记录，并自动派发首条任务。
