# 启用图像绘制 Agent

‍

默认团队里的「**复杂图像绘制 Agent**」负责把研究进展和结论转成图——架构图、曲线、论文风格的示意图，并把产物写回 Research Blackboard 供其他成员引用。

## 操作

- 在团队面板选中「**复杂图像绘制 Agent**」成员标签。
- 它的「主 Skill」已自动绑定 **research-image-agent**。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/tut22-image.jpg)

- 如不需要绘图成员，可在其标签上点 ✕ 移除；需要时再用「添加」补回，主 Skill 选 **research-image-agent** 即可恢复绑定。
