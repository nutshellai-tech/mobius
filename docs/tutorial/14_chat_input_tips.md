# 聊天输入小技巧

‍

> 会话输入栏里藏着几个高频小功能，能让你更顺手地驱动智能体。下面 4 个按钮都在输入框那一行：

![image](https://serve.nutshellai.cn/publish/auto/tutorial/14_tips_01_toolbar.jpg)

## ① 发送（加急）：打断当前输出，立即发送

闪电图标就是**发送（加急）**。当智能体正在长篇输出、你想立刻插话或改方向时，点它——当前输出会被打断，你的新消息立即送出，不必等它说完。

## ② 找回之前输入的指令（回放输入）

历史图标是**回放输入**。点开能看到本会话历史发过的所有指令，支持**搜索、回放、复制**——找回之前的 prompt 不用重新打一遍。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/14_tips_02_replay.jpg)

## ③ 语音输入

麦克风图标是**语音输入**。点一下开始录音，说完自动转文字填进输入框；录音中再点一次（方块图标）即可停止。

## ④ 手动压缩上文

点「**+**」菜单 → **压缩上文**，把较长的对话压成摘要，省 token、延长上下文窗口。压缩期间你**仍可继续发指令**（响应会延后），也可点「终止」打断。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/14_tips_03_compress.jpg)

‍

> 一句话：闪电加急、历史回放、麦克风口述、加号压缩——输入栏四个键，效率翻倍。
