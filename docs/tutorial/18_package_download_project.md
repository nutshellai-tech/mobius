# 小技巧：项目文件打包下载

‍

需要把整个项目的代码 / 文件导出成压缩包带走？项目页内置了**打包下载**。

## 怎么做

1. 进入项目，顶部标签切到 **「打包下载」**。
2. 勾选要打包的文件 / 文件夹（可全选），点 **「下载」**，系统打包成 zip 并开始下载。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/tip-package.jpg)

## 注意

- 默认只列**项目绑定目录下的表层文件和文件夹**。
- 工作缓存目录（`.imac` / `.mobius`）默认**不选**；即便手动勾上，系统也会跳过其中的 `package_zip`，避免把生成中的包再打进去。
- 包很大时（默认阈值 500MB）会二次确认，生成和下载可能稍慢。
