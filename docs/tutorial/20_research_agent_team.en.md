# Build an Agent Research Team

‍

Inside a research task, click "New Agent" on the right → "Create Agent Team" to open the team panel. It ships with 3 default members (1 chief + 2 assistants); each is created and auto-started in turn, and the right side shows a 3D team board.

![image](https://serve.nutshellai.cn/publish/auto/tutorial/tut20-team.jpg)

## Add / remove members

- **Add**: click the green "**Add**" button at the end of the tab strip to add an assistant (max 12).
- **Remove**: hover an assistant tab and click the ✕ that appears.
- **The chief cannot be deleted**: every team needs exactly one chief to decompose the topic and integrate conclusions.

## Each member is independently configurable

- Name, purpose / responsibility, model, language, main Skill, and which Skills / Memories are in scope.
- Click "**Create & Start Team**" and the system creates each session, writes a team record to the Research Blackboard, and dispatches the first task automatically.
