# 小莫：强大的 Agent 操作系统控制中枢

‍

> **小莫（Xiaomo）** 是莫比乌斯的智能助理，也是整个 Agent 操作系统的**控制中枢**。你只需用一句话告诉它目标，它就能感知当前项目、调度其他智能体去开发任意功能，甚至直接修改莫比乌斯自身、完成自我迭代。本篇带你走完一遍。

### 1. 调整小莫模型和配置（可跳过）

打开小莫的**预设配置**，可以选用什么模型、什么语言、什么性格，以及启用哪些技能和记忆。不调整也能直接用——小莫会按系统默认配置工作。

### 2. 询问当前的项目和近期进展

不必自己翻页面——直接问小莫「**当前有哪些项目、最近在做什么**」，它会汇总各项目和执行会话的近况，给你一份清晰的进展总览。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/13_xiaomo_control_hub_04.jpg)

### 3. 指示小莫，让小莫指挥其他智能体，开发任意项目任意功能

把目标交给小莫，例如「**在 XX 项目里实现 YY 功能**」。小莫会自动拆解任务、创建任务单与执行会话，并**调度对应的智能体**去真正编码、验证、交付——你只管提需求。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/13_xiaomo_control_hub_05.jpg)

### 4. 指示小莫对 Mobius 系统自身修改，进行自我迭代

小莫还能**修改莫比乌斯本身**：告诉它「给系统加一个 ZZ 能力 / 修一个 Bug」，它会在自迭代项目里创建受控任务、改动代码并重新部署——这就是莫比乌斯的**自我进化**。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/13_xiaomo_control_hub_06.jpg)

‍

> 一句话总结：小莫是**指挥官**——问它进展、派它干活、让它改造系统，都只需一句话。
