# 小技巧：修改前端色彩主题

‍

右上角有一个**主题按钮**，点一下就能切换界面配色（浅色 / 深色 / 多款预设主题）。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/tip-theme.jpg)

## 用法

- **单击**：在当前主题的浅 / 深之间切换。
- **Alt + 单击**（Mac 上 Option + 单击）：切到**下一款**预设主题（按钮 title 会提示「Alt+点击切换下一个主题」）。
- 当前主题名就显示在按钮上（如「深色」）。

> 主题选择会记在你的浏览器里，换台设备需要重选。
