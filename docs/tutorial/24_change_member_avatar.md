# 更换成员形象

‍

团队配置面板右侧是一块 3D 画布，每个成员都站在舞台上。你可以用顶部工具栏的「**形象**」下拉，给整个团队换一套造型——纯属视觉呈现，不影响 Agent 行为。

## 操作

- 团队面板右上角的「**形象**」下拉提供 7 种造型：方块机器人 / 晶体核心 / 双螺旋 / 宇航机器人 / 水母 / 蘑菇 / 企鹅。
- 旁边的「**场景**」下拉可切换 9 种背景（城市天际线 / 研究实验室 / 深空星港 …）。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/tut24-avatar.jpg)

!!! tip
    形象和场景只是展示层；首席（chief）始终用绿色、助理用蓝色，选中态会高亮，方便在画布上直接点选正在编辑的成员。
