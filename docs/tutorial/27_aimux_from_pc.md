# 从个人工作站或者笔记本电脑连接 Mobius 系统

个人工作站或者笔记本电脑当然也可以直接连接 Mobius 系统，主要方式有两种：

- 通过 **aimux + python 命令行**连接
- 通过**桌面客户端**连接

本文主要介绍第一种。

## 第 1 步：打开连接指引

![连接指引面板](https://serve.nutshellai.cn/publish/auto/tutorial/tut27-connect-guide.jpg)

## 第 2 步：点击复制按钮，复制第一条安装指令

复制 **(1)**。

![复制按钮](https://serve.nutshellai.cn/publish/auto/tutorial/tut27-copy-btn.jpg)

## 第 3 步：打开命令行，开始安装，粘贴刚才的命令，开始安装

![打开命令行](https://serve.nutshellai.cn/publish/auto/tutorial/tut27-terminal.jpg)

![开始安装](https://serve.nutshellai.cn/publish/auto/tutorial/tut27-installing.jpg)

## 第 4 步：复制连接命令，粘贴到命令行

这一次，复制 **(2)**。

![复制按钮](https://serve.nutshellai.cn/publish/auto/tutorial/tut27-copy-btn.jpg)

![粘贴连接命令](https://serve.nutshellai.cn/publish/auto/tutorial/tut27-connect-cmd.jpg)
