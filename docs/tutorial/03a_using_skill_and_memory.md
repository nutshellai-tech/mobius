# 创建并使用技能（Skill）和记忆（Memory），并与同事分享

‍

## 莫比乌斯系统中的 Skill 和 Memory 分为以下三种

- **内置**：部署项目时就有的常用 Skill

  - 作用于所有人，所有项目
- **用户级** Skill 和 Memory：在用户主页添加和编辑，作用于**所有**项目

  - 用户级的 Skill 和 Memory 默认情况下**只有自己**能看到和使用
- **项目级** Skill 和 Memory：在项目主页添加和编辑，作用于**某个**项目

  - 例如一个调试Python代码的技能，不应该作用于一个JavaScript前端项目，不然浪费Token也容易引发模型的困惑
  - 项目级 Skill 和 Memory 可以被所有**有权访问项目**的人使用
