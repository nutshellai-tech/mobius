# 小技巧：中途切换模型

‍

一个会话跑到一半，想换另一个模型接着干（比如换更强的模型啃硬骨头、或换便宜的跑收尾）——不用重开，**「修改模型并继续」** 会基于当前上下文派生一个新会话。

## 怎么做

1. 在执行会话页面，点聊天框旁边的 **「更多操作」（⋯）** 菜单。
2. 选 **「修改模型并继续」**。
3. 在弹出的对话框里挑一个新模型，确认即可。

![image](https://serve.nutshellai.cn/publish/auto/tutorial/tip-model-switch.jpg)

## 注意

- 它会**新建一个会话**（名字带「- 更换模型」后缀），把当前上下文带过去，而不是原地改模型——这样原会话保持完整，可回溯。
- 适合：长任务里按阶段换模型（构思用强模型、批量执行用快模型）。
