# 从外部导入已有项目或者代码片段，并开启第一个任务

### 1. 从本地上传

1. 新建一个项目

![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628200556-zozifyt.png)

2. 输入模型名称，修改权限让所有人可见

![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628201020-ouea61b.png)

3. 选择【在 vscode 中打开】

![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628201112-pqazsb1.png)

4. 把一个或者多个文件拖动到vscode server中

![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628201540-am01jko.png)

5. 创建Issue（任务），告诉智能体：【**使用刚上传的文件（zip）创建项目，并初始化Git版本控制**】，随后**一路确定**

![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628201800-7l4b9i6.png)![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628202028-6drxfu7.png)![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628202044-wacr7c1.png)

![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628202054-wvmk19t.png)

6. 开始第一个任务：输入【**跑一个3D粒子群优化的简单demo，然后绘制轨迹，展示优化过程。把绘制好的图片展示给用户！** 】

	- **选择 I**（在刚才会话中**继续**提问）

![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628202456-tdx322c.png)

	- **选择 II**（创建新的会话，和刚才的会话**并行**运行，互不干涉）

![image](https://assets.b3logfile.com/siyuan/1779383706239/assets/image-20260628202701-j445ct9.png)

7. 喝杯茶
8. 等一会，然后检查结果
